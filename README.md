# rock-paper-scissors
*The Odin Project *

Rock paper scissors game in html and JavaScript

> ## Skills:
> - basic HTML
> - basic JavaScript

## Sections
* r = rock
* p = paper
* s = scissors
- Player input (computer and user)
- Game status:
    - win (r > s > p > r)
    - tie
